# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Multi-tab snooze: select multiple tabs and snooze them together as a group
- Snooze options are now fully configurable in Settings (add, edit, delete, reorder)
- Added "1 Hour from Now" and "2 Hours from Now" snooze options
- Snooze panel redesigned as a vertical list with wakeup times shown inline

### Changed
- Snooze panel buttons are now text-only (icons removed) for a cleaner look
- Keyboard navigation updated for vertical list layout
